Check-In Semanal

Para realizar um Check-in das mídias, siga os passos abaixo:

  • 1. Selecione o painel que deseja realizar o check-in;
  • 2. Após isso, todas as mídias vinculadas a esse painel serão exibidas;
  • 3. Junto delas, terá um espaço para anexar as fotos, que será o comprovante de exibição de cada mídia;
  • 4. Após anexar todas as fotos, é só clicar em 'Enviar Check-In'.
